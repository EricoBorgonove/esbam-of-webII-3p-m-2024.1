Propriedades de afastamento

m - margem - afastamento do elemento da borda (ou seja, o espaço entre a borda e o conteúdo)
p - padding - preenchimento interno do elemento (espaco dentro dele próprio)
border - linha que rodeia todo o elemento
Propriedades de posição relativa
t - top parte superior do elemento
b - botton - parte inferior do elemento
s - start - parte inicial do elemento
e - end - parte final do elemento
x - posição relativa ao eixo x - inclui os valores start e end
y - posição relativa ao eixo y - inclui os valores top e botton
tamanhos relativos
vai de 0 até 5
onde 0 é completamente sem e 5 é o maior de todos
p-0 até p-5 // px-5 // p-5 // pt-5 //
m-0 ate m-5 // mt-3 // mx-2 // my-0
